The naming convention of the generated graph files:

ER-(x)v-(y)n-(z).NET
SF-(x)v-(y)n-(z).NET

ER = Erdos
SF = Scale Free
v = vertex
n = neighbours (average)
x = the number of vertices in the graph
y = the average number of edges per vertex (Degrees)
z = instance of this graph configuration (1 to 3)
.NET = file extension
